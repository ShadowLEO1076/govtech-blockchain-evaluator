///Uso del comando de evaluación docente
Pasos
1_Escribe el comando especial en el chat de Discord:
!menu
2_Selecciona la opción correspondiente y arrastra el archivo JSON de evaluación docente al canal o ventana del bot.
3_ Envía el archivo. 

El sistema procesará automáticamente la información y mostrará los resultados de la evaluación.

////Requisitos
Tener acceso al servidor de Discord.
Contar con el archivo JSON de evaluación docente generado previamente tenemos evaluaciones docente de prueba en el directorio de evaluaciones .
Utilizar el comando !menu antes de enviar el archivo
